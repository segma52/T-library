Это небольшая библиотека. Открывается без текстового редактора, после первого запуска создает в папке с программой 2 файла с счетчиком и данными о книгах.
Я старался сделать интерфейс максимально красивым, понятным и удобным пользователю.
слева от лупы находится поиск, книги ищутся сразу после ввода букв
"+" - добавление книги, после нажатия появляется меню с описанием книги
внизу есть фильтры по распространенным жанрам и сортировка книг по 3-м признакам
так же книги можно скроллить как сайты и удалять
